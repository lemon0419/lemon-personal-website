## Visual Language
- 核心视觉签名：手绘线条插画风格笔触的边框和装饰元素，如手绘装饰线和简化的图形；所有边框不严格，微微偏移的手绘笔触小圆角
- 材质与深度：无深度影、纯面平，背景使用纸张纹理或单色，不使用渐变
- 容器与按钮：圆角容器，无影子，细框边；按钮使用Primary填充，引导视觉焦点
- 布局节奏：完整大面积留白，内容区域稀疏有致；元素之间使用宽松散的间距，保持简约

## Animation
- 入场：开场页面第插画小人接待页的淡入效果；点击后拉开幕布的动画效果切换到主页面；主页面元素的逍遥淡入
- 交互：按钮悬停时微微缩小和透明度变化
- 过渡：日夜模式切换和中英文切换时使用CSS transition，过渡时间0.3s ease-in-out；幕布拉开效果使用纯CSS keyframes实现

## Forbidden
- 禁止使用大渐变和毛玻璃效果
- 禁止使用混色模式或过度混色的CSS filter
- 禁止模板化的网格背景或复杂图案纹理

## Additional Notes
- 所有用户可见文案为中文和英文版本
- 插画风格头像使用第一张图片（如有用户提供）或绘制简约手绘线条人物细束
- 手绘颜色笔触无特殊要求的地方不画如，仅笔述笔述心的笔触整体即可
